# 入门指南

## 安装

尝试 SWC 最简单的方式是使用[在线演示平台](/playground)。

或者，运行以下命令下载预编译的二进制文件：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>
    ```bash
    pnpm add -D @swc/cli @swc/core
    ```
  </Tabs.Tab>
    <Tabs.Tab>
    ```bash
    npm i -D @swc/cli @swc/core
    ```
  </Tabs.Tab>
    <Tabs.Tab>
    ```bash
    yarn add -D @swc/cli @swc/core
    ```
  </Tabs.Tab>
</Tabs>

然后，你可以转译你的第一个文件并输出到`stdout`：

```bash
npx swc ./file.js
```

## 支持的二进制版本

SWC 可以作为预编译的二进制文件下载使用，也可以从源代码构建。目前，提供以下二进制版本：

- Mac (Apple Silicon)
- Mac (x64)
- Linux (x86_64)
- Linux (aarch64)
- Linux (armv7)
- Alpine Linux (同时安装 `@swc/core-linux-musl`)
- Android (aarch64)
- Windows (win32-x64)
- Windows (ia32)
